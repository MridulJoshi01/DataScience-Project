# DataScience-Project

A collection of data science projects showcasing analysis, visualization, and insights from various datasets.

## 📁 Repository Structure

### Major Project
Analysis of COVID-19 pandemic waves in India with comprehensive visualizations and insights.

- **COVID_India_Wave_Comparison.csv** - Dataset containing COVID-19 statistics across different waves in India
- **Main Project.pbix** - Power BI dashboard for interactive visualization of COVID-19 trends
- **Main Project.pdf** - Detailed project report with findings and analysis

### Mini Project
Multiple data analysis projects exploring e-commerce and entertainment datasets.

- **MiniProject.ipynb** - Jupyter notebook with consolidated data analysis
- **project.ipynb** - Additional analysis notebook
- **flipkart.csv** - Flipkart e-commerce dataset for product and sales analysis
- **netflix_titles.csv** - Netflix titles dataset for entertainment analytics

## 🎯 Project Highlights

### Major Project: COVID-19 India Wave Comparison
This project analyzes and compares different waves of the COVID-19 pandemic in India, providing insights into:
- Wave trends and progression
- Statistical comparisons across waves
- Interactive Power BI visualizations
- Data-driven insights and patterns

### Mini Project: Multi-Dataset Analysis
Exploration and analysis of:
- **Flipkart E-commerce Data** - Product analysis, pricing trends, ratings, and sales patterns
- **Netflix Titles Data** - Content analysis, genre distribution, release trends, and platform insights

## 🛠️ Technologies & Tools

- **Jupyter Notebook** - Data analysis and exploration
- **Power BI** - Interactive dashboard creation and visualization
- **Python** - Data processing and analysis
- **CSV** - Data storage and exchange
