# DataScience-Project 📊

A comprehensive collection of data science projects showcasing analysis, visualization, and insights from various datasets. Built primarily with **Jupyter Notebooks** for interactive data exploration and analysis.

## 🎯 Overview

This repository contains multiple data science projects ranging from pandemic analysis to e-commerce and entertainment data exploration. Each project demonstrates data cleaning, exploratory data analysis (EDA), statistical analysis, and visualization techniques.

### 🔴 Major Project: COVID-19 India Wave Comparison

A comprehensive analysis of COVID-19 pandemic waves in India with detailed statistical comparisons and interactive visualizations.

**Files:**
- `COVID_India_Wave_Comparison.csv` - COVID-19 statistics dataset across different waves in India
- `Main Project.pbix` - Interactive Power BI dashboard
- `Main Project.pdf` - Detailed project report with findings

**Analysis Includes:**
- Wave trends and progression patterns
- Statistical comparisons and correlations
- Data-driven insights and patterns
- Interactive visual dashboards

### 🟢 Mini Project: Multi-Dataset Analysis

Hands-on exploration and analysis of real-world datasets with practical insights.

**Notebooks:**
- `MiniProject.ipynb` - Consolidated data analysis notebook
- `project.ipynb` - Additional analysis and experiments

**Datasets:**
- `flipkart.csv` - E-commerce dataset including product details, pricing, ratings, and sales trends
- `netflix_titles.csv` - Entertainment dataset with content details, genres, and release information

**Analysis Focus:**
- **Flipkart E-commerce Analysis:** Product categories, pricing trends, rating distributions, seller insights
- **Netflix Content Analysis:** Genre distribution, release trends, content growth over time, platform insights

Additional coursework and practice exercises demonstrating fundamental data science concepts.

## 🚀 Getting Started

### Prerequisites
- Python 3.7+
- Jupyter Notebook
- pandas, numpy, matplotlib, seaborn

### Installation

```bash
# Clone the repository
git clone https://github.com/MridulJoshi01/DataScience-Project.git
cd DataScience-Project

# Install required packages
pip install jupyter pandas numpy matplotlib seaborn

# Launch Jupyter Notebook
jupyter notebook
```

### Running the Projects

1. **Major Project:** Review the PDF report and open the Power BI dashboard for interactive exploration
2. **Mini Project:** Open the `.ipynb` notebooks in Jupyter for step-by-step analysis
3. **Assignments:** Execute assignment notebooks for practice exercises
